<TableOfContents />

## Overview

Update a specific custom accounting field by ID.

<RequestExample>
```bash
curl \
  -X PUT \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "display_name": "Updated Department Name",
    "is_required": true
  }' \
  'https://api.ramp.com/developer/v1/accounting/fields/field_1234567890'
```
</RequestExample>

## Path Parameters

<ParamField path="id" type="string" required>
  The ID of the custom field to update.
</ParamField>

## Request Fields

<ParamField body="display_name" type="string">
  New display name for the custom field.
</ParamField>

<ParamField body="is_required" type="boolean">
  Whether the field should be required for transactions.
</ParamField>

{
  "data": {

</ResponseExample>

## Response Fields

<ResponseField name="data" type="object">
  The updated custom accounting field.
</ResponseField>

<ResponseField name="data.created_at" type="string<date-time>">
  The date and time the field was created.
</ResponseField>

<ResponseField name="data.display_name" type="string">
  User-friendly display name for the field.
</ResponseField>

<ResponseField name="data.id" type="string">
  The unique identifier for the field.
</ResponseField>

<ResponseField name="data.input_type" type="string">
  Type of input field.
</ResponseField>

<ResponseField name="data.is_required" type="boolean">
  Whether the field is required for transactions.
</ResponseField>

<ResponseField name="data.is_splittable" type="boolean">
  Whether the field can be split across multiple transactions.
</ResponseField>

<ResponseField name="data.name" type="string">
  Internal name for the field.
</ResponseField>

<ResponseField name="data.ramp_id" type="string<uuid>">
  ID to uniquely identify a custom field within Ramp system.
</ResponseField>

<ResponseField name="data.updated_at" type="string<date-time>">
  The date and time the field was last updated.
</ResponseField>

<Note>
  **OAuth scopes required:** `accounting:write`
</Note>

## Examples

### JavaScript

```javascript
const fieldId = 'field_1234567890';
const response = await fetch(`https://api.ramp.com/developer/v1/accounting/fields/${fieldId}`, {
  method: 'PUT',
  headers: {
    'Accept': 'application/json',
    'Authorization': `Bearer ${RAMP_API_TOKEN}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    display_name: 'Updated Department Name',
    is_required: true
  })
});

const data = await response.json();
console.log(data.data);
```

### Python

```python
import requests
import json

field_id = 'field_1234567890'
headers = {
    'Accept': 'application/json',
    'Authorization': f'Bearer {RAMP_API_TOKEN}',
    'Content-Type': 'application/json'
}

payload = {
    'display_name': 'Updated Department Name',
    'is_required': True
}

response = requests.put(
    f'https://api.ramp.com/developer/v1/accounting/fields/{field_id}',
    headers=headers,
    data=json.dumps(payload)
)

data = response.json()
print(data['data'])
```
