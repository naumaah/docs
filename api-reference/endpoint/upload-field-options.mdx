## Overview

Upload new options for a custom accounting field.

<RequestExample>
```bash
curl \
  -X POST \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "field_id": "46910cc3-ab41-4b80-b4a7-94dab9f1b795",
    "options": [
      {
        "id": "418",
        "value": "Engineering & Design"
      },
      {
        "id": "514",
        "value": "Sales & Marketing"
      }
    ]
  }' \
  'https://api.ramp.com/developer/v1/accounting/field-options'
```
</RequestExample>

## Request Fields

<ParamField body="field_id" type="string<uuid>" required>
  The ID of the custom field to add options to.
</ParamField>

<ParamField body="options" type="array<object>" required>
  Array of option objects to create.
</ParamField>

<ParamField body="options[].id" type="string" required>
  Unique identifier for the option.
</ParamField>

<ParamField body="options[].value" type="string" required>
  Display value for the option.
</ParamField>

<ParamField body="options[].code" type="string" required>
  Code of the custom accounting field option; e.g. 400-100.
</ParamField>

## Response Fields

<ResponseField name="options" type="array<object>">
  Array of created field options.
</ResponseField>

<ResponseField name="options[].code" type="string">
  Code of the custom field option; Usually it is a unique identifier for the custom field option in the remote ERP system.
</ResponseField>

<ResponseField name="options[].created_at" type="string<date-time>">
  The date and time the option was created.
</ResponseField>

<ResponseField name="options[].id" type="string">
  The unique identifier for the option.
</ResponseField>

<ResponseField name="options[].display_name" type="string">
  Display name of the custom field option.
</ResponseField>

<ResponseField name="options[].ramp_id" type="string<uuid>">
  ID to uniquely identify a custom field option within Ramp system.
</ResponseField>

<ResponseField name="options[].updated_at" type="string<date-time>">
  The date and time the option was last updated.
</ResponseField>

<Note>
  **OAuth scopes required:** `accounting:write`
</Note>

## Examples

### JavaScript

```javascript
const response = await fetch('https://api.ramp.com/developer/v1/accounting/field-options', {
  method: 'POST',
  headers: {
    'Accept': 'application/json',
    'Authorization': `Bearer ${RAMP_API_TOKEN}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    field_id: '46910cc3-ab41-4b80-b4a7-94dab9f1b795',
    options: [
      {
        id: '418',
        value: 'Engineering & Design'
      },
      {
        id: '514',
        value: 'Sales & Marketing'
      }
    ]
  })
});

const data = await response.json();
console.log(data.data);
```

### Python

```python
import requests
import json

headers = {
    'Accept': 'application/json',
    'Authorization': f'Bearer {RAMP_API_TOKEN}',
    'Content-Type': 'application/json'
}

payload = {
    'field_id': '46910cc3-ab41-4b80-b4a7-94dab9f1b795',
    'options': [
        {
            'id': '418',
            'value': 'Engineering & Design'
        },
        {
            'id': '514',
            'value': 'Sales & Marketing'
        }
    ]
}

response = requests.post(
    'https://api.ramp.com/developer/v1/accounting/field-options',
    headers=headers,
    data=json.dumps(payload)
)

data = response.json()
print(data['data'])
```
