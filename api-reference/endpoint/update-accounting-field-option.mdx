title: 'PUT: Update a custom accounting field option'

method: 'PUT'

---

<TableOfContents />

## Overview

Update a specific custom accounting field option by ID.

<RequestExample>
```bash
curl \
  -X PUT \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "display_name": "Updated Engineering & Design"
  }' \
  'https://api.ramp.com/developer/v1/accounting/field-options/opt_1234567890'
```
</RequestExample>

## Path Parameters

<ParamField path="id" type="string" required>
  The ID of the custom field option to update.
</ParamField>

## Request Fields

<ParamField body="display_name" type="string">
  New display name for the custom field option.
</ParamField>

## Response

<ResponseExample>
```json
{
  "data": {
    "code": "418",
    "created_at": "2021-01-01T00:00:00Z",
    "id": "opt_1234567890",
    "display_name": "Updated Engineering & Design",
    "ramp_id": "46910cc3-ab41-4b80-b4a7-94dab9f1b795",
    "updated_at": "2021-01-01T12:00:00Z"
  }
}
```
</ResponseExample>

## Response Fields

<ResponseField name="data" type="object">
  The updated custom field option details.
</ResponseField>

<ResponseField name="data.code" type="string">
  Code of the custom field option; Usually it is a unique identifier for the custom field option in the remote ERP system.
</ResponseField>

<ResponseField name="data.created_at" type="string<date-time>">
  The date and time the option was created.
</ResponseField>

<ResponseField name="data.id" type="string">
  The unique identifier for the option.
</ResponseField>

<ResponseField name="data.display_name" type="string">
  Display name of the custom field option.
</ResponseField>

<ResponseField name="data.ramp_id" type="string<uuid>">
  ID to uniquely identify a custom field option within Ramp system.
</ResponseField>

<ResponseField name="data.updated_at" type="string<date-time>">
  The date and time the option was last updated.
</ResponseField>

<Note>
  **OAuth scopes required:** `accounting:write`
</Note>

## Examples

### JavaScript

```javascript
const optionId = 'opt_1234567890';
const response = await fetch(`https://api.ramp.com/developer/v1/accounting/field-options/${optionId}`, {
  method: 'PUT',
  headers: {
    'Accept': 'application/json',
    'Authorization': `Bearer ${RAMP_API_TOKEN}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    display_name: 'Updated Engineering & Design'
  })
});

const data = await response.json();
console.log(data.data);
```

### Python

```python
import requests
import json

option_id = 'opt_1234567890'
headers = {
    'Accept': 'application/json',
    'Authorization': f'Bearer {RAMP_API_TOKEN}',
    'Content-Type': 'application/json'
}

payload = {
    'display_name': 'Updated Engineering & Design'
}

response = requests.put(
    f'https://api.ramp.com/developer/v1/accounting/field-options/{option_id}',
    headers=headers,
    data=json.dumps(payload)
)

data = response.json()
print(data['data'])
```
