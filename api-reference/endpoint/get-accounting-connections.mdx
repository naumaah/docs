## Overview

Fetch all accounting connections for the current business.

<RequestExample>
```bash
curl \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  'https://api.ramp.com/developer/v1/accounting/all-connections'
```
</RequestExample>

## Response Fields

<ResponseField name="connections" type="array<object>">
  A list of accounting connections for the current business.
</ResponseField>

<ResponseField name="connections[].created_at" type="string<date-time>">
  The date and time the accounting connection was created.
</ResponseField>

<ResponseField name="connections[].id" type="string<uuid>">
  The unique identifier for the accounting connection.
</ResponseField>

<ResponseField name="connections[].is_active" type="boolean">
  Whether the accounting connection is active.
</ResponseField>

<ResponseField name="connections[].last_linked_at" type="string<date-time> or null">
  The date and time the accounting connection was last connected.
</ResponseField>

<ResponseField name="connections[].remote_provider_name" type="string or null">
  The name of the accounting connection.
</ResponseField>

<ResponseField name="connections[].status" type="string">
  The current status of the accounting connection.
  
  **Allowed values:**
  - `failed_to_auth` - Authentication failed
  - `linked` - Successfully linked
  - `revoked` - Access was revoked
  - `unlinked` - Connection was unlinked
</ResponseField>

## Authentication

This endpoint requires authentication using a Bearer token.

<Note>
  **OAuth scopes required:** `accounting:read`
</Note>

## Examples

### JavaScript

```javascript
const response = await fetch('https://api.ramp.com/developer/v1/accounting/all-connections', {
  method: 'GET',
  headers: {
    'Accept': 'application/json',
    'Authorization': `Bearer ${RAMP_API_TOKEN}`
  }
});

const data = await response.json();
console.log(data.connections);
```

### Python

```python
import requests

headers = {
    'Accept': 'application/json',
    'Authorization': f'Bearer {RAMP_API_TOKEN}'
}

response = requests.get(
    'https://api.ramp.com/developer/v1/accounting/all-connections',
    headers=headers
)

data = response.json()
print(data['connections'])
```
