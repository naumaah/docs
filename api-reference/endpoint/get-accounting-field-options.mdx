## Overview

List options for a given custom accounting field. This endpoint allows you to retrieve all available options for a specific custom field that has been created in your accounting system.

<RequestExample>
```bash
curl \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  'https://api.ramp.com/developer/v1/accounting/field-options'
```
</RequestExample>

## Query Parameters

<ParamField query="field_id" type="string" required>
  The ID of the custom field to get options for.
</ParamField>

<ParamField query="remote_id" type="string">
  Filter by Remote/external ID of custom accounting field options in ERP system.
</ParamField>

<ParamField query="code" type="string">
  Filter by code of custom accounting field options.
</ParamField>

<ParamField query="start" type="string">
  The ID of the last entity of the previous page, used for pagination to get the next page.
</ParamField>

<ParamField query="page_size" type="integer">
  The number of results to be returned in each page. The value must be between 2 and 100. If not specified, the default value 20 will be used.
</ParamField>

<ParamField query="is_active" type="boolean">
  Whether or not value is active.
</ParamField>

## Response Fields

<ResponseField name="data" type="array<object>">
  A list of options for the specified custom field.
</ResponseField>

<ResponseField name="data[].code" type="string<uuid>">
  Code of the custom field option; Usually it is a unique identifier for the custom field option in the remote ERP system.
</ResponseField>

<ResponseField name="data[].created_at" type="string<date-time>">
  The date and time the option was created.
</ResponseField>

<ResponseField name="data[].id" type="string">
  The unique identifier for the option.
</ResponseField>

<ResponseField name="data[].display_name" type="string">
Display name of the custom field option.
</ResponseField>

<ResponseField name="data[].ramp_id" type="string<uuid">
  id to uniquely identify a custom field option within Ramp system
</ResponseField>

<ResponseField name="data[].updated_at" type="string<date-time>">
  The date and time the option was last updated.
</ResponseField>

## Authentication

This endpoint requires authentication using a Bearer token.

<Note>
  **OAuth scopes required:** `accounting:read`
</Note>

## Examples

### JavaScript

```javascript
const fieldId = 'field_1234567890';
const response = await fetch(`https://api.ramp.com/developer/v1/accounting/field-options?field_id=${fieldId}`, {
  method: 'GET',
  headers: {
    'Accept': 'application/json',
    'Authorization': `Bearer ${RAMP_API_TOKEN}`
  }
});

const data = await response.json();
console.log(data.options);
```

### Python

```python
import requests

field_id = 'field_1234567890'
params = {
    'field_id': field_id
}

headers = {
    'Accept': 'application/json',
    'Authorization': f'Bearer {RAMP_API_TOKEN}'
}

response = requests.get(
    'https://api.ramp.com/developer/v1/accounting/field-options',
    params=params,
    headers=headers
)

data = response.json()
print(data['options'])
```

### cURL

```bash
curl \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  'https://api.ramp.com/developer/v1/accounting/field-options?field_id=field_1234567890'
```

```bash

  'https://api.ramp.com/developer/v1/accounting/field-options?field_id=field_1234567890&remote_code=ENG'
```
