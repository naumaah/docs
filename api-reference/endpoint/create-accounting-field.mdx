## Overview

Create a new custom accounting field.

<RequestExample>
```bash
curl \
  -X POST \
  -H "Accept: application/json" \
  -H "Authorization: Bearer $RAMP_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": "Department",
    "input_type": "SINGLE_CHOICE",
    "is_splittable": true,
    "name": "Department",
    "display_name": "Department"
  }' \
  'https://api.ramp.com/developer/v1/accounting/fields'
```
</RequestExample>

## Request Fields

<ParamField body="id" type="string" required>
  Unique identifier for the field.
</ParamField>

<ParamField body="input_type" type="string" required>
  Type of input field.
  
  **Allowed values:**
  - `SINGLE_CHOICE` - Single selection dropdown
  - `MULTIPLE_CHOICE` - Multiple selection options
  - `TEXT` - Free text input
  - `BOOLEAN` - True/false checkbox
</ParamField>

<ParamField body="is_splittable" type="boolean" required>
  Whether the field can be split across multiple transactions.
</ParamField>

<ParamField body="name" type="string" required>
  Internal name for the field.
</ParamField>

<ParamField body="display_name" type="string">
  User-friendly display name for the field.
</ParamField>

## Response Fields

<ResponseField name="data" type="object">
  The created custom accounting field.
</ResponseField>

<ResponseField name="data.created_at" type="string<date-time>">
  The date and time the field was created.
</ResponseField>

<ResponseField name="data.display_name" type="string">
  User-friendly display name for the field.
</ResponseField>

<ResponseField name="data.id" type="string">
  The unique identifier for the field.
</ResponseField>

<ResponseField name="data.input_type" type="string">
  Type of input field.
</ResponseField>

<ResponseField name="data.is_required" type="boolean">
  Whether the field is required for transactions.
</ResponseField>

<ResponseField name="data.is_splittable" type="boolean">
  Whether the field can be split across multiple transactions.
</ResponseField>

<ResponseField name="data.name" type="string">
  Internal name for the field.
</ResponseField>

<ResponseField name="data.ramp_id" type="string<uuid>">
  ID to uniquely identify a custom field within Ramp system.
</ResponseField>

<ResponseField name="data.updated_at" type="string<date-time>">
  The date and time the field was last updated.
</ResponseField>

<Note>
  **OAuth scopes required:** `accounting:write`
</Note>

## Examples

### JavaScript

```javascript
const response = await fetch('https://api.ramp.com/developer/v1/accounting/fields', {
  method: 'POST',
  headers: {
    'Accept': 'application/json',
    'Authorization': `Bearer ${RAMP_API_TOKEN}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 'Department',
    input_type: 'SINGLE_CHOICE',
    is_splittable: true,
    name: 'Department',
    display_name: 'Department'
  })
});

const data = await response.json();
console.log(data.data);
```

### Python

```python
import requests
import json

headers = {
    'Accept': 'application/json',
    'Authorization': f'Bearer {RAMP_API_TOKEN}',
    'Content-Type': 'application/json'
}

payload = {
    'id': 'Department',
    'input_type': 'SINGLE_CHOICE',
    'is_splittable': True,
    'name': 'Department',
    'display_name': 'Department'
}

response = requests.post(
    'https://api.ramp.com/developer/v1/accounting/fields',
    headers=headers,
    data=json.dumps(payload)
)

data = response.json()
print(data['data'])
```
